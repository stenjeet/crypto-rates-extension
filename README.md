# Crypto Rates — Chrome Extension

A Google Chrome extension for quickly viewing current cryptocurrency rates.

## Features

- Display of current cryptocurrency rates
- List of **favorite coins** on the main screen
- **“+” button** to open the full list of available coins
- Add coins to favorites
- Remove coins from favorites
- Search by coin
- Clean and simple UI

## How to Use

1. Install the extension in Google Chrome
2. Click the extension icon in the browser toolbar
3. Your favorite coins are displayed on the main screen
4. Click the **“+”** button to open the full list of coins
5. Add coins to your favorites
6. Remove coins from favorites when needed


## Technologies

- React
- TypeScript
- Chrome Extensions API

## Installation (Developer Mode)

1. Clone the repository:
   ```
   git clone <repo-url>
   ```

2. Install dependencies:
```
npm install
```

3. Run locally:
```
npm run dev
```

4. Build:
``` 
npm run build
```

5. Open chrome://extensions/
6. Enable Developer mode
7. Click Load unpacked
8. Select the project folder
